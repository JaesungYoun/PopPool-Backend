## 💡 서비스 소개
**흩어져 있는 팝업스토어 정보를 제공하고 인스타그램 연동으로 부가 기능을 제공하는 앱 서비스**
>PopPool은 PopUpStore + Pool 의 약자로, 팝업의 장이라는 의미의 이름입니다.</br>
>PopPool은 다양한 팝업스토어들에 대해 소개하고 인스타그램 연동으로 교류할 수 있는 서비스를 제공합니다.
